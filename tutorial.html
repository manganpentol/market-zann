Tutorial Penggunaan ZannStore

Selamat datang di halaman tutorial ZannStore. Di sini Anda akan mempelajari cara menggunakan berbagai fitur yang tersedia dalam website dan aplikasi ini.

🔧 1. Tools Termux

Pilih salah satu tools dari menu seperti Vbug, Virus Link, Spam Call, dll. Setelah itu tekan tombol "Salin Git Clone", lalu buka Termux dan tempelkan perintah tersebut dengan menekan lama dan pilih "Paste".

🛍️ 2. Produk Digital

Masuk ke halaman Produk. Di sini tersedia layanan seperti:

  • Panel Pterodactyl
  • Sc Bug
  • Murband

Pilih produk, hubungi admin via WhatsApp yang tersedia, dan lakukan pembayaran.

🖥 3. Panel

Kunjungi halaman Panel untuk melihat List Harga Panel dan layanan yang bisa langsung digunakan. Tersedia juga tombol "Salin Perintah" agar mudah dikloning ke Termux.

📚 4. Tutorial & Edukasi

Di halaman ini kamu akan temukan:

  • Cara menggunakan Termux dari nol
  • Cara konek VPS
  • Cara install script panel
  • Tips & Trik buging aman

✅ Tips Umum Penggunaan:

  • Pastikan koneksi internet stabil
  • Gunakan Termux terbaru dari F-Droid
  • Jangan jalankan lebih dari satu tools bersamaan
  • Selalu baca deskripsi sebelum menyalin perintah
